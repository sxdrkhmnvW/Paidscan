Paidscan — это мощный многофункциональный CLI-инструмент, разработанный на Python для автоматизации задач в сфере кибербезопасности, сетевого сканирования и OSINT (разведки по открытым источникам). Утилита объединяет более 38 инструментов в одном удобном интерактивном терминальном интерфейсе.
🚀 Основные возможности

Инструмент разбит на логические модули для удобства навигации:

    Сканирование: Высокоскоростное сканирование портов хоста, обнаружение активных устройств в сети, сравнение результатов сканирования.

    Сеть & DNS: Пинг с графиком, трассировка маршрута (Traceroute), сбор DNS/WHOIS/ASN информации, чтение ARP-таблиц.

    Веб & OSINT: Определение веб-технологий (HTTP), анализ SSL-сертификатов, поиск субдоменов, парсинг Email на сайтах, работа с Google Dorks, Username Hunter и поиск по Wayback Machine.

    Геолокация: Определение текущего местоположения, геолокация по IP, определение производителя по MAC-адресу.

    Безопасность & Атаки: Модули для тестирования на проникновение (Брутфорс SSH/FTP/HTTP/MySQL, поиск директорий, сканеры SQLi, XSS, LFI, стресс-тестирование и проверка паролей по базам утечек HIBP).

📥 Установка и запуск

    💡 Примечание: Инструмент тестировался и отлично работает на Linux (Fedora/Ubuntu). Для работы требуется Python версии 3.6 или выше.

1. Клонирование репозитория

Склонируйте проект к себе на локальную машину:
Bash

git clone [https://github.com/sxdrkhmnvW/Paidscan.git](https://github.com/sxdrkhmnvW/Paidscan.git)
cd Paidscan

2. Установка зависимостей

Установите все необходимые библиотеки Python из файла требований:
Bash

pip install -r requirements.txt

3. Запуск утилиты

Запустите основной скрипт программы:
Bash

python Paid/Paidscan.py

⚙️ Настройка (Опционально)

Для использования некоторых продвинутых модулей (например, проверки утечек паролей или специфических OSINT-сервисов), вы можете настроить свои ключи API внутри файла конфигурации paidscan_config.json или через переменные окружения в .env (шаблоны игнорируются git и не попадут в сеть).
